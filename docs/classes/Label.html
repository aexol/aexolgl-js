Label Class

Extends GameObject
Module: Font

Label class creates single line text located in 3d space

  • font
  • size
)

Parameters:

  • scene Scene

    scene

  • string String

    string for single line text

  • font String

    font to use( Rememeber about loading fonts to document with @font-face )

  • size Integer

    font size
